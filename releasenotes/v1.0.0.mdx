## 🔹 Overview 
This release introduces a powerful new Sermon Search tool.<br />
###### [v1.0.0 Source Code](https://github.com/secc/Numa/releases/tag/v1.0.0)

## 🚀 New Features
- AI-powered semantic search allows users to converse with past sermons.
- Users can switch between sermon collections within a single conversation.
- New sermons can be uploaded to sermon collections by users w/ upload permissions.
- Admin settings can be changed by admin users to customize the app's look & feel.
